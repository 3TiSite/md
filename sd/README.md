## مارڪ ڊائون ترجمو

مارڪ ڊائون ترجمو آزاديءَ سان استعمال ڪري سگھجي ٿو يا گھڻن ٻولين واري جامد سائيٽ جنريٽر سان جلدي ويب سائيٽ ٺاهڻ لاءِ.
